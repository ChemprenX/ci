案例征集

奖项介绍

“金网奖”创办于2009年，由金网汇及天极传媒集团主办。八载沉淀，其已成为集评奖、颁奖盛典、国内外嘉宾高峰对话、创新展、网络营销白皮书、大咖聚会、年度系列线上及线下沙龙等为一体的营销品牌，是网络营销领域最具国际化与本土化的权威奖项。

金网奖作为最具互联网特质和精神的网络营销奖项，在新消费环境下，致力于推动人、生活与商业的完美融合，构建新连接关系。以人为中心，契合热点及用户情绪，运用LBS、场景、大数据等技术手段，组合运用多元化媒介，从创意、策略到执行，实现营销的触达、互动、效果的完美统一，推动智慧新生活、未来新商业的达成。

金网奖诚邀所有实效营销创意案例加盟评奖。所有参赛案例均由国内顶尖营销精英团队进行多轮评审，决出获奖案例，联合百家媒体进行海量传播。通过优秀营销人物、创新的营销产品及工具、处于引领地位的企业品牌，来起到示范作用，推动营销在生活、商业中的落地。

参赛须知

  • 1、日程安排

    报送时间：2017年2月24日—7月31日
    颁奖盛典：2017年9月

  • 2、参赛要求

    无论是营销策略整合，还是能够达成目标效果的单一策略，任何网络营销传播的成果都可以角逐金网奖各项大奖。这意味着，不管是运用了最新的LBS、场景、大数据等技术，还是契合行业热点及把握用户情绪，或是选择最活跃、接地气的媒介组合，任何达成营销的触达、互动、效果完美统一的网络营销成功案例，都可以参赛 报送的参赛案例需满足两个条件：一，必须是在2016年7月31日至报送截止日期间实施并投放过的；二，必须已取得一定的成果，拥有可量化的传播效果（大部分相关数据必须在规定的投放时间内获取）。

  • 3、参赛流程

    详细阅读“第九届金网奖案例征集参赛信息说明”，关注官方微信公众号“金网奖”，及时了解金网奖最新动态。

    1）案例包装

    B、案例包装：

    3）完成支付


    A、 报名时间与参赛费用：
    案例类奖项：
    4月30日前报名：2000/奖项案例
    5月31日前报名：2500/奖项案例
    7月31日前报名：3000/奖项案例
    非案例类TOP榜单：
    5月31日前报名：8000/奖项
    7月31日前报名：10000/奖项

    B、 支付方式：
    支付方式一：在网站支付页面通过支付宝完成付款；
    支付方式二：转账至指定账户。

    C、完成审核：
    付款完成后，将支付截图及发票信息发至指定邮箱：wangjl@imcc.org.cn
    回复邮件标题格式为：公司名称+案例报名费+发票类型
    发票项目：服务费
    发票类型：国税增值税普通发票、国税增值税专用发票（开专用票必须提供开票信息、一般纳税人证明资料）

    邮件发送后的2个工作日内，组委会将以Email方式通知参赛单位核实相关信息。如未收到回复，请致电 010-88856008-717查询作品提交情况。

    4）提供授权


    案例审核通过后，需通过邮寄方式，将“2017年第九届金网奖授权书”寄送至“金网奖”组委会备案。

    A、“金网奖”组委会及案例作品征集部将保留改变参赛作品投递奖项类别的权利。

    B、“金网奖”案例征集、案例收费、费用支出等明细将全程保持财务透明。

奖项分类与设置

第九届金网奖共设置“案例类”与“非案例类” 两大类别奖项。
案例类奖项设有场景、智能、原生、体育、公益、泛娱乐六大类别，每大类下设立小类别“专项奖”，每类“专项奖”评选出金、银、铜奖；最终从金奖作品中评选出一件“全场大奖”。
非案例类榜单设有人物、品牌、产品 三大类别，根据具体细分类别分别评选入围TOP榜单，排行榜将根据排名进行综合排行。

案例类奖项


1、场景类

场景营销类:
将营销行为与人们的生活场景结合，进行广告植入或展示的营销手段，往往通过用户行为、数据分析，精准性更好，效果转化更高。
移动营销类:
利用移动设备（手机或平板电脑）为主要传播渠道，在移动网络环境，在移动应用嵌入广告或借助LBS位置信息进行等营销手段，向目标受众定向和精确地传递个性化即时信息，激发消费者广泛参与，达到一对一互动营销目的的案例。
大屏营销类:
通过互联网电视大屏，针对客厅场景下的用户需求，实现品牌创意、内容的深度定制，创新用户交互形式，帮助广告主从展示广告走向品商合一，进行高效营销。
电商O2O类:借助线上互动或促销活动、数据挖掘等手段，激发消费者实时购买行为并在销量数据上获得突破的案例；结合用户的生活场景，通过打折、提供信息、服务预订等线上方式，吸引用户线下消费或到店获取服务的营销案例。
2、智能类

智能营销类:
将人工智能、深度学习、机器学习等新技术融入品牌营销，在交互、场景、营销全程实现智能化，从多渠道洞察用户，更好地激发品牌与用户连接。
大数据营销类:
基于多平台的大量用户多维数据，依托大数据技术基础，进行精准、定向的广告信息传递的营销手段，可按人群、时间、区域等程序式投放，最终给品牌企业带来更高的营销投资回报的案例。
AR/VR营销类:
在营销中通过VR/AR技术，或语音识别、图像识别等创新交互形式，增强趣味性、重度沉浸感体验，在用户交互、信息传递形式具有颠覆性的变化，更好地表现广告创意，让用户充分参与其中，营销效果及转化率表现更为突出。
适用于多屏环境，采用创新的3D、VR等新兴交互体验技术，或在营销方案执行中，创新技术起到了关键引导或创意表现的营销案例。

3、原生类

情感营销类:
以“人”情感为纽带进行创意发挥，更加注重人文层面、情感层面的需求满足。广告创意、内容契合用户群情绪和社群价值观，在保障用户体验的同时，通过整合多样化媒介资源，传递匹配的产品及品牌信息的营销案例。
社会化营销类:
利用微博、微信或交互页面，发起趣味性、参与度高的营销活动，重点体现内容营销和社交关系链的分享、创作机制，实现产品或品牌信息的最大化扩散的营销活动
4、体育类

体育营销类:
以体育赛事为载体来推广自己的产品和品牌的一种市场营销活动案例，综合运用视频、社会化传播载体，强调品牌与产品与体育精神的契合，传递品牌理念。奖项设置该类别的唯一大奖。
5、公益类

公益营销类：
利用互联网以关心人的生存发展、社会进步为出发点，借助公益活动与消费者进行沟通，在产生公益效果的同时，使消费者对企业的产品或服务产生偏好，并由此提高品牌知名度和美誉度的案例；奖项设置分金奖、银奖和铜奖。
健康营销类：
在营销的过程中，对营销对象进行精准定位，采取无“污染”的营销方法，避免营销中的资源浪费、避免对无关客户的骚扰以及不负责任采用搜索引擎作弊的方式进行推广营销。
6、娱乐类

直播营销类：
营销活动以直播平台为载体，精准定位用户，在现场随着事件的发生、发展进程同时制作和播出节目的播出方式，与用户实时互动，达到品牌提升或销量增长的目的。
娱乐营销类：
借助娱乐的元素或形式，一般为影视剧、明星、音乐会及衍生线下活动等，将产品、品牌信息以创新内容融入到娱乐媒介上，实现与目标受众互动，从而达到提升品牌货销售产品目的的营销方式。
短视频营销类：
企业和品牌主借助于短视频这种媒介形式，结合品牌调性、用户属性和营销战役做社会化营销，实现病毒式传播，垂直细分领域从娱乐八卦、段子延伸到汽车、时尚、美食、体育等。

非案例类奖项


1、人物类

年度网络营销领军人物：
中国互联网营销领域发展的推动者和见证者，他们促进了网络营销模式、方法的进化，带来了更创新、更具实效的网络营销实践,将中国互联网营销推向高峰。
年度网络营销新锐人物：
他们是全营销各领域勇于突破和挑战的先锋人物，是推动中国互联网营销领域发展的新锐力量，用敏锐的目光、雷厉风行的执行力为中国互联网营销领域带来新的创新和变革。
2、品牌类

年度网络营销最佳品牌主 ：
洞察最新营销趋势，利用互联网营销手段、营销工具、选择多样的创意形式及合理媒体投放的品牌主。
3、产品类

年度网络营销优秀产品及工具：
最大限度利用互联网创新技术，其产品对互联网营销产生影响与推动。
4、排行榜

年度数字营销代理机构排行榜：
在互联网营销领域，发挥创意、技术驱动及资源整合等数字化综合传播能力，实现品牌价值及销售转化的数字营销机构，将最终通过综合实力评估，评选出整合发展力年度排行榜。
