关于 about

“金网奖”创办于2009年，由金网汇及天极传媒集团主办。八载沉淀，其已成为集评奖、颁奖盛典、国内外嘉宾高峰对话、创新展、网络营销白皮书、大咖聚会、年度系列线上及线下沙龙等为一体的营销品牌。作为最具互联网特质和精神的网络营销奖项，金网奖在新消费环境下，致力于推动人、生活与商业的完美融合。
