闪耀20年

20年来，网络营销发生了哪些变化？

1、营销进入用户主权时代，一切不以人为中心的营销都收效甚微

2、场景化、生活化成为营销的发展趋势

红铜时代

从1996--2000年，印刷广告一统天下的时代，随着互联网的出现被打破。网络营销开始萌芽，传统广告开始向网络广告过度，门户网站Banner广告、邮件营销等初期的网络营销方式出现在大众视线里。

关键节点:

（1）1997年，中国第一支来自于知名品牌IBM的网络广告在比特网投放。（植入比特）

（2）门户广告Banner广告（新浪、网易、搜狐）

青铜时代

2001--2005年，步入新世纪，伴随互联网的高速发展，营销人开始探索各种新的网络营销方式。网络广告表现形式多样化，搜索营销萌芽并初步长成“小巨人”，社交时代开启。在这个蓬勃发展的时代，孕育着生机和一切可能。

关键节点：

（1）百度搜索营销（植入百度）

（2）社交时代开启，Facebook、开心网、人人网火爆

黄金时代

2006--2010年，网络广告进入高速发展的轨道，视频广告等各种花样百出的富媒体形式让人们的眼球眼花缭乱、传统的展示型广告开始更加注重人的参与和互动体验，社会化媒体蓬勃发展开启了新的营销时代、社会化营销受到认可和广泛关注。这是一个任营销海阔天空畅游的黄金时代。

关键节点：

（1）社会化营销案例

（2）富媒体营销形式

（3）与芙蓉姐姐、奶茶妹妹、凤姐等网络红人逐一跳出屏幕互动

白金时代

2011至今，迎来了一个更加炫目的白金时代，用户行为转向移动互联网，移动广告全面爆发，电商营销兴起，O2O大爆炸，程序化广告崛起，社会化营销走红。新技术和创意无时无刻不在出现，网络营销全面爆发。

关键节点：

（1）电商营销

（2）程序化平台

（3）移动广告
